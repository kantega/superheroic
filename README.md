# Superheroic Web Development with AngularJS #

## About ##

Workshop for learning about [AngularJS](http://angularjs.org/) at [Booster](http://www.boosterconf.no/) 2014.

## Usage ##

__Maven__

        mvn clean install
        mvn jetty:run

__Gradle__

        ./gradlew (Linux/Mac) || gradlew.bat (Windows)
        gradle jettyEclipseRun
